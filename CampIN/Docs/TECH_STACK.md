# CampIn — Tech Stack

## Frontend
- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS (utility-first)
- **Icons:** Lucide React
- **Animations:** Framer Motion (smooth page transitions and hover effects)

## Backend & Services
- **Auth/Database:** Supabase (PostgreSQL + Auth)
- **Forms:** Tally Forms (for MVP lead capture)
- **Storage:** Supabase Storage or Cloudinary (for optimized listing photos)
- **Payments:** Razorpay (India-specific gateway)

## Infrastructure
- **Hosting:** Vercel (CI/CD integrated)
- **Analytics:** GA4 + Microsoft Clarity (Heatmaps)
- **Communications:** Resend (Transactional emails)
