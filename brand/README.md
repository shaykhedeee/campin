# CampIn Brand Asset System

## Core Mark

Primary logo:

`brand/campin-logo.svg`

Use for website nav, decks, proposals, and public documents.

## Icon Variants

Monochrome:

`brand/campin-icon-monochrome.svg`

Use for stamps, single-color documents, invoices, and low-color contexts.

Social avatar:

`brand/campin-social-avatar.svg`

Use for Instagram, WhatsApp, X, LinkedIn, and newsletter profile images.

## Trust Badges

CampIn Reviewed:

`brand/campin-reviewed-badge.svg`

Use only after basic manual review: location, host contact, photos, and core amenities.

Founding Host:

`brand/founding-host-badge.svg`

Use for first host partners who pass the founding host onboarding process.

Road Stop Standard:

`brand/road-stop-standard-badge.svg`

Use only after parking, washroom, water, and local support are checked.

## Color Values

- Forest Green: `#1F3D2B`
- Warm Orange: `#E67E22`
- Off White: `#F7F5F2`
- Sky Mist: `#DCE9E1`
- Text Dark: `#2D2D2D`

## Usage Rules

- Do not use "Verified" until the stronger checklist is complete.
- Do not use "official" unless a government or tourism body has formally approved it.
- Do not place badges on unreviewed candidate leads.
- Do not stretch or recolor the logo outside this palette.
- Keep trust badges small, clear, and close to the claim they support.
