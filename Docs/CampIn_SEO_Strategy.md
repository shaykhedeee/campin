## 4. TECHNICAL SEO CHECKLIST

### On-Page
- [ ] Unique title tag per page (60 chars max)
- [ ] Unique meta description per page (155 chars max)
- [ ] Open Graph images for all pages
- [ ] H1 tag on every page (only one per page)
- [ ] Proper heading hierarchy (H1 → H2 → H3)
- [ ] Alt text on all images
- [ ] Internal linking between related pages
- [ ] Schema markup for listings (LocalBusiness)

### Technical
- [ ] XML sitemap submitted to Google Search Console
- [ ] robots.txt properly configured
- [ ] SSL certificate (HTTPS)
- [ ] Page speed < 3 seconds on mobile
- [ ] Mobile responsive (Core Web Vitals pass)
- [ ] Canonical URLs on all pages
- [ ] 301 redirects for any changed URLs

### Local SEO
- [ ] Google Business Profile for CampIn
- [ ] Location-based landing pages
- [ ] Local business schema markup
- [ ] Reviews on Google Maps (Phase 2)
