# CampIn Grievance Redressal And Support Policy

Draft version: 2026-05-21

## Contact

General support:

hello@campin.co.in

## Support Categories

- Booking or pilot issue.
- Refund or cancellation issue.
- Host concern.
- Safety incident.
- Community moderation issue.
- Privacy or data request.
- Incorrect listing or verification claim.
- Partnership inquiry.

## Response Targets

| Issue Type | Target First Response |
|---|---:|
| Safety incident during stay | As soon as possible |
| Active pilot/arrival issue | 2 hours |
| Refund/cancellation issue | 2 business days |
| Privacy/data request | 7 business days |
| Community moderation issue | 2 business days |
| Listing correction | 3 business days |

## Required Complaint Fields

- Name.
- Email/phone.
- Issue type.
- Booking/pilot reference if any.
- Date.
- Location/listing if any.
- Description.
- Screenshots/photos where relevant.
- Desired resolution.

## Resolution Process

1. Acknowledge issue.
2. Create issue record.
3. Classify severity.
4. Contact host/camper/partner if needed.
5. Review evidence.
6. Decide outcome.
7. Communicate resolution.
8. Log learning in Trust Matrix.

## Severity Levels

| Level | Meaning | Action |
|---|---|---|
| S1 | Immediate physical safety issue | Emergency action, founder review |
| S2 | Active stay failure or harassment | Same-day investigation |
| S3 | Refund, amenity mismatch, host dispute | Evidence review |
| S4 | General support or correction | Normal queue |

## Marketplace Launch Note

Before CampIn becomes a transactional marketplace, the grievance contact, escalation process, refund workflow, and legally required consumer disclosures should be reviewed by counsel.
